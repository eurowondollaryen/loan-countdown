# 대출 상환 카운트다운 계산기 (Loan Countdown Calculator)

이 앱은 사용자의 월 수익, 지출 및 저축 전략을 바탕으로 대출 상환까지 남은 기간을 계산해주는 크로스플랫폼 앱입니다.

## 주요 기능
- **상환 기간 계산**: 월 고정 수익에서 지출을 제외한 금액으로 대출을 갚아나갈 때 걸리는 개월수를 계산합니다.
- **중도상환 시뮬레이션**: 특정 금액(임계치)이 쌓였을 때만 중도상환을 하는 시나리오를 시뮬레이션합니다.
- **상환 상태 표시**: 남은 개월수에 따라 상/중/하로 나누어 시각적으로 표시합니다.

## 개발 스택
- **Framework**: Flutter (Dart)
  - 이유: 가장 대중적인 크로스플랫폼 프레임워크이며, 단일 코드베이스로 iOS, Android, Web 환경을 모두 지원합니다.
- **상태 관리**: `StatefulWidget` (단일 페이지 앱으로 간단한 상태 관리 적용)

## 상환 상태 기준 (Status Thresholds)
남은 기간에 따라 다음과 같이 상태를 분류합니다:
- **상 (Low Duration)**: 12개월 미만 (녹색 표시)
- **중 (Medium Duration)**: 12개월 ~ 36개월 (주황색 표시)
- **하 (High Duration)**: 36개월 초과 (빨간색 표시)

## 상환 로직 설명
1. 매월 `수익 - 지출`을 계산하여 '저축 항아리(Pot)'에 담습니다.
2. 저축 항아리의 금액이 '중도상환 임계금액' 이상이 되거나, 남은 대출금을 모두 갚을 수 있는 수준이 되면 즉시 상환합니다.
3. 대출금이 0이 될 때까지 소요된 총 개월수를 반환합니다.
   - 수익이 지출보다 적어 상환이 불가능한 경우 에러 메시지를 표시합니다.

## 결정 사항 및 가이드
- **이자율 미포함**: 현재 요구사항에는 이자율에 대한 언급이 없으므로, 원금 상환만을 기준으로 계산합니다. (고정 지출에 이미 대출 이자가 포함되어 있다고 가정할 수 있습니다.)
- **중도상환 임계치**: 사용자가 '저축 금액이 얼마 쌓이면 중도상환할지'를 입력하도록 하여, 실제 사용자의 저축 습관을 반영할 수 있도록 하였습니다.
- **상환 상태 기준**: 1년(상), 3년(중)을 기준으로 잡았습니다. 이는 일반적인 신용대출 상환 기간을 참고한 수치입니다.
